# Mapa Mental: W3C (World Wide Web Consortium)

## ¿Qué es la W3C?

- Organización internacional que establece estándares técnicos para la Web.
- Fundada por Tim Berners-Lee para garantizar interoperabilidad y crecimiento a largo plazo.

## Misión

- Desarrollar protocolos, convenciones y guías que permitan que la Web funcione para todos.
- Promover interoperabilidad, accesibilidad, usabilidad y apertura.

## Tipos de estándares que desarrolla la W3C

- Lenguajes (HTML5, XML)
- Hojas de estilo (CSS)
- Accesibilidad (WCAG, WAI-ARIA)
- Multimedia (WebAudio, WebRTC)
- Gráficos y visualización (SVG, WebGL)
- APIs para apps web (Fetch, Service Workers)

## ¿Quiénes participan?

- Grupos de trabajo conformados por miembros (empresas, universidades, ONGs) y participantes públicos.
- W3C Staff/Directores, miembros con veto y contribuyentes.

## Importancia para desarrolladores web

- Garantiza interoperabilidad entre navegadores y plataformas.
- Mejora accesibilidad y la experiencia de usuario.
- Provee guías de buenas prácticas y APIs confiables.

## Ejemplos/Relaciones

- HTML5 + CSS = Estructura y diseño (base del frontend)
- WCAG + WAI-ARIA = Accesibilidad
- SVG = Gráficos escalables y animables sin pixelar

## Notas adicionales

- W3C tiene documentos como Recomendaciones (W3C Recommendations), Working drafts y Candidate Recommendations.
- Los estándares son desarrollados via consenso y pasan por procesos de revisión pública.
